## AUDIT & PERBAIKAN

### 1. GUI VERIFIKASI
- Bandingkan layout, warna, font, icon, jarak, ukuran.
- Semua tombol dan tab sama persis dengan bobot2.py.
- Semua komponen GUI aktif dan terhubung ke fungsi backend yang benar.

### 2. FITUR PRE-START
Sebelum bot mulai, user bisa:
- Menentukan TP (pips, harga, %, mata uang)
- Menentukan SL (pips, harga, %, mata uang)
- Memilih strategi awal
- Mengatur lot size atau auto-lot
- Mengaktifkan/menonaktifkan news filter
Semua setting ini wajib tersimpan dan dipakai strategi ketika bot dijalankan.

### 3. INTEGRASI SISTEM
- Pastikan status MT5 real-time di GUI.
- Pastikan semua setting user langsung diterapkan ke strategi.
- Pastikan auto-reconnect MT5 bekerja.

### 4. PERBAIKAN ERROR & OPTIMISASI
- Zero LSP, null reference, atau runtime error.
- Validasi semua input user.
- Optimalkan performa GUI (min CPU/RAM).
- Logging untuk semua perubahan setting.

### 5. STARTER SCRIPT .BAT
Buat file `start_bot.bat` yang:
- Mengaktifkan virtual environment (jika ada)
- Menjalankan script utama bot (`python bobot2.py` atau nama script baru)
- Menampilkan log langsung di terminal
Contoh format dasar:
